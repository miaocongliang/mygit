 C，删除远程仓库（比如GitHub）的目录或文件

具体操作

拉取远程的Repo到本地（如果已经在本地，可以略过） 
$ git clone xxxxxx
在本地仓库删除文件 
$ git rm 我的文件
在本地仓库删除文件夹 
$ git rm -r 我的文件夹/
此处-r表示递归所有子目录，如果你要删除的，是空的文件夹，此处可以不用带上-r。
提交代码 
$ git commit -m"我的修改"
推送到远程仓库（比如GitHub） 
$ git push origin xxxxxx
补充: git rm

查看git rm的说明文档:

$ git rm -h

用法：git rm [<选项>] [--] <文件>...

    -n, --dry-run         演习

    -q, --quiet           不列出删除的文件

    --cached              只从索引区删除

    -f, --force           忽略文件更新状态检查

    -r                    允许递归删除

    --ignore-unmatch      即使没有匹配，也以零状态退出
